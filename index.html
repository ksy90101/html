HTML History

Hyper Text Markup Language

HTML의 역사를 살펴보겠습니다.

HTML1.0

HTML 1.0

  • 1991년에 팀 버나스리가 발표한 최초의 HTML이다.
  • 당시에 웹 사이트 제작에 참여한 사람이 많지 않았으며, 언어가 매우 제한적이었습니다.
  • 표현할 수 있는 데이터도 굉장히 제한적이였으며, 거의 텍스트 위주의 정보전달 목적으로 사용되었다.
HTML2.0

HTML 2.0

  • 1995년에 국제 표준으로 제정된 최초의 HTML
  • 1.0에서 파일 업로드 양식과 프레임, 테이블, 이미지맵, 국제화 기능이 추가되었다.
  • 1997년 1월까지 웹사이트 디자인의 표준이 되었고, 처음으로 많은 핵심 HTML 기능들이 정의되었다.
HTML3.2

HTML 3.2

  • 1997년에 W3C에 의해 제정된 최초의 HTML
  • 수학 수식을 사용하는 태그를 완전히 제거하고, 넷스케이프의 비주얼 관련 태그(b, font)가 들어갔다.
HTML4.01

HTML 4.01

  • 1999년에 안정된 표준의 버전으로 나왔다.
  • CSS로 디자인적인 요소를 구분하게 되고, HTML로는 웹페이지의 전반적인 구조만을 명시하는 것을 원칙으로 하게 된다.
  • W3C에서는 이제 HTML은 다양화된 웹페이지를 모두 표현하기에 기술적인 한계에 부딛혔다고 판단하고, 더이상 HTML을 발전시키지 않을 것을 공표하게 된다.
HTML5

HTML5

  • 2007년에 WHATWG에서 정의한 표준을 따르기로 하면서 나왔다.
  • 지금의 HTML5와 CSS3이 표준으로 확정되었다.
  • 시맨틱 태그, 멀티미디어 태그, 폼 관련 태그들이 추가되었다.
